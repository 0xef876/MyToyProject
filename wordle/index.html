Game Rule

Guess the word!

  • If alphabet is in the word, and position is correct, It's background color is Yellow.
  • If alphabet is in the word, and position is different, It's background color is Green.
  • If alphabet is not in the word, It's background color is Gray.

단어를 맞춰보세요!

  • 맞추려는 단어와 입력한 알파벳과 위치가 같다면, 배경색은 노란색입니다.
  • 맞추려는 단어에 입력한 알파벳은 있지만, 위치가 다르다면, 배경색은 초록색입니다.
  • 맞추려는 단어에 입력한 알파벳이 없다면, 배경은 회색입니다.
